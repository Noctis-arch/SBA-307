ShopInCart Website Project by Marlon R

Description

ShopInCart is a simple three-page website designed to simulate an online store. The site allows users to browse products, learn about the store, and submit contact or feedback forms. The goal of this project was to create a clean, beginner-friendly website while meeting all required HTML and CSS specifications.

Pages Included

1. Home (index.html): Displays featured products, images, a GIF, and a dropdown menu for sorting.
2. About (about.html): Provides information about the store and includes a product comparison table.
3. Contact (contact.html): Contains two forms for user input (customer information and feedback).

Technical Specifications

HTML Requirements

1. Includes three separate pages with consistent navigation between them.
2. Uses more than ten HTML tags, including:
html, head, body, header, nav, section, footer, h1-h3, p, img, table, tr, th, td, form, input, textarea, button, select, option
3. Contains one table for product comparison.
4. Includes two forms for user input.
5. Features a dropdown menu using the <select> tag.
6. Incorporates text, images, and a GIF for content variety.

CSS Requirements

1. Uses external CSS via a linked stylesheet (styles.css).
2. Includes internal CSS within the <style> tag.
3. Uses inline CSS for specific elements.
4. Applies at least five different CSS selectors (element, class, etc.).
5. Implements Flexbox to organize product layout on the home page.
6. Uses a consistent color scheme across all pages.
7. Includes two CSS animations:
8. A fade/slide animation on product cards
9. A hover scaling effect on buttons
